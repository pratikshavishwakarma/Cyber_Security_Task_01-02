PART A – CIA TRIAD RESEARCH

This section explores the three foundational pillars of information security.

Confidentiality

Ensures that information is accessible only to authorized individuals through encryption, access controls, authentication mechanisms, and least-privilege principles.

Integrity

Ensures that information remains accurate, complete, and protected from unauthorized modification through hashing, digital signatures, checksums, and audit logs.

Availability

Ensures that systems, services, and information remain accessible to authorized users whenever required through redundancy, backups, failover mechanisms, and disaster recovery planning.

Case Studies Referenced

• Equifax Data Breach (2017)

• SolarWinds Supply Chain Attack (2020)

• Colonial Pipeline Ransomware Attack (2021)

PART E – SECURITY RECOMMENDATIONS

Based on the risk assessment and threat analysis, several prioritized recommendations were developed.

Key Recommendations

• Enable Multi-Factor Authentication (MFA)

• Implement Formal Patch Management

• Deploy Endpoint Detection and Response (EDR)

• Conduct Security Awareness Training

• Enforce Strong Password Policies

• Implement Network Segmentation

• Establish Backup and Recovery Procedures

• Configure Email Security Controls

• Implement Role-Based Access Control (RBAC)

• Conduct Regular Security Assessments

Expected Benefits
