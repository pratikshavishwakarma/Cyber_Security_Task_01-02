PROJECT OVERVIEW

This repository contains the complete submission for Cyber Security Task 01 – Personal Security Audit & Cyber Awareness Assessment.

The project focuses on evaluating the security posture of a Windows system, reviewing password and account security practices, researching common cyber threats, developing cybersecurity awareness material, and reflecting on personal security improvements.

The objective is to demonstrate foundational cybersecurity knowledge, security assessment techniques, threat awareness, and technical documentation skills.

PART B – PASSWORD SECURITY REVIEW

This section examines password security principles and best practices for protecting digital accounts.

Topics Covered

• Strong password characteristics

• Password complexity requirements

• Password reuse risks

• Credential stuffing attacks

• Multi-Factor Authentication (MFA)

• Personal password security policy

Recommended Practices

• Use passwords with at least 14 characters

• Enable Multi-Factor Authentication

• Use unique passwords for every account

• Store credentials in a password manager

• Avoid predictable passwords and personal information
